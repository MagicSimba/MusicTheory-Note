根据不同的调式，一个黑键即可以是<font color=red>升音</font>，又可以是<font color=red>降音</font>

![](../assets/images/%E9%BB%91%E9%94%AE%E9%9F%B3%E5%90%8D.png)

## 升/降符号

基本的升降号有：`♯` 升号，`♭` 降号

因黑键已经带有符号，且不能带有两个符号，所以又衍生出 `𝄪` 重升号，`𝄫` 重降号

!!! note
    C=♯B=𝄫D
