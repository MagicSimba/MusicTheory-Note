## 自然大调

当音阶遵循 <font color=red>全全半全全全半</font> 的规则，就称之为自然大调，简称为大调

![](../assets/images/staff/大调式.png){ width="70%" }


当调号为 <font color=red>1 = C</font> 时，这个大调就称为 <font color=red>C大调</font>

调式必须保证每一个音名都出现一次，根据不同的情况需求，每个键即可以是<font color=red>升音</font>，又可以是<font color=red>降音</font>

<font color=red>C 大调:</font> C D E F G A B C

<font color=red>♯C 大调:</font> ♯C ♯D ♯E ♯F ♯G ♯A ♯B ♯C

<font color=red>D 大调:</font> D E ♯F G A B ♯C

<font color=red>♭D 大调:</font> ♭D ♭E F ♭G ♭A ♭B C ♭D

### 等音调

声音相同但音名不同的大调，称为等音调

例如 <font color=red>♯C 大调</font> 和 <font color=red>♭D 大调</font>，二者弹奏出来的完全相同，但是表示方法却不相同
